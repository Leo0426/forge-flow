# 治理能力

`skills/governance/` 存放项目治理相关的能力，包括诊断、任务管理和仓库健康检查。

## 当前能力

- [`diagnose-lite/`](diagnose-lite/SKILL.md)：轻量级诊断流程，用最小复现、假设、验证、修复和再验证快速定位 bug、错误或性能问题。
- [`repo-health/`](repo-health/SKILL.md)：从维护者视角检查仓库健康度，包括 ownership、依赖、安全、文档新鲜度、架构漂移和质量门禁。
- [`issue-breakdown/`](issue-breakdown/SKILL.md)：使用 tracer bullet 垂直切片，把 PRD、计划或设计拆成可独立验证的工程任务。
