# 杂项能力

`skills/misc/` 存放不属于特定工程阶段的横切能力，主要用于沟通压缩、能力创建与项目初始化。

## 当前能力

- [`terse-communication/`](terse-communication/SKILL.md)：中英双语极简沟通模式，压缩冗余，保留技术准确性与结构完整性。
- [`personal-skill-builder/`](personal-skill-builder/SKILL.md)：创建、优化和评审结构化 agent skills，输出符合 ForgeFlow 模式的 SKILL.md 和可选参考文档。
- [`setup-forgeflow/`](setup-forgeflow/SKILL.md)：在新仓库中搭建 ForgeFlow agent runtime 上下文，包括 issue tracker、triage labels 和领域文档布局。
