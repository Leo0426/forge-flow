# 产品能力

`skills/product/` 存放产品发现、需求梳理与反馈处理相关的能力。

## 当前能力

- [`discovery/`](discovery/SKILL.md)：在进入 PRD 前验证问题、用户、替代方案和 MVP 成功指标，用于个人项目的 0-1 价值判断。
- [`to-prd/`](to-prd/SKILL.md)：将想法、对话上下文或粗略需求整理成轻量 PRD，适合个人开发者和企业功能澄清场景。
- [`feedback-to-roadmap/`](feedback-to-roadmap/SKILL.md)：将 MVP 结果、用户反馈、bug 和复盘结论整理成下一轮 roadmap。
