# 工程能力

`skills/engineering/` 存放日常编码工作相关的能力。

## 当前能力

- [`architecture/`](architecture/SKILL.md)：识别并改进代码库中的架构复杂度，使用 Deep Module、最小接口、隐藏变化和分层一致性提出重构候选。
- [`zoom-out/`](zoom-out/SKILL.md)：上升一层抽象，给出当前代码区域的系统地图：相关模块、调用者、依赖方向，使用项目领域词汇。
- [`code-review/`](code-review/SKILL.md)：对当前 diff 做 correctness、简化/复用和方法论对齐三类审查，关注正确性 bug 和值得改的结构问题。
- [`coding-with-tdd/`](coding-with-tdd/SKILL.md)：使用测试驱动开发的红绿重构循环实现功能或修复 bug，用 tracer bullet 垂直切片推进 MVP。
- [`git-commit/`](git-commit/SKILL.md)：从当前变更中提炼最小一致性提交，并生成高质量 git commit。
